# Tianlei Zhang Personal Website

This repository hosts my academic personal website:

https://ynbsztl.github.io

The site is built with Jekyll and GitHub Pages, based on the Academic Pages /
Minimal Mistakes template. The current repository is organized around the
published website, search tools, CV files, and supporting data.

## Website Sections

- **Home / About**: personal introduction, research interests, education, and
  contact information. Source: `_pages/about.md`.
- **Research**: publications and research-related content. Source:
  `_pages/publications.md`.
- **Econ Search**: English economics paper search based on static JSON shards.
  Source page: `_pages/econ-search.md`; frontend script:
  `assets/js/econ-search-static.js`; generated data:
  `assets/econ-search/`.
- **CNKI Search**: Chinese economics paper search for selected Chinese journals.
  Source page: `_pages/cnki-search.md`; frontend script:
  `assets/js/cnki-search-static.js`; generated data:
  `assets/cnki-search/`.
- **Curriculum Vitae**: PDF CV and LaTeX source files. Main PDF:
  `ztlcv/CV_Tianlei.pdf`.
- **Optional / hidden pages**: data, talks, miscellaneous materials, sitemap,
  and 404 pages live under `_pages/`.

The main navigation bar is configured in `_data/navigation.yml`.

## Project Structure

- `_config.yml`: global Jekyll and site configuration.
- `_pages/`: website pages and page-level HTML/CSS.
- `_data/`: navigation, author metadata, and theme text.
- `_includes/`, `_layouts/`, `_sass/`: Jekyll theme components.
- `assets/`: CSS, JavaScript, fonts, and generated search indexes.
- `files/`: downloadable files and source CSVs used to generate search indexes.
- `images/`: website images.
- `scripts/`: data-processing scripts for search index generation.
- `ztlcv/`: CV PDF and LaTeX source files.
- `_miscellany/`: archived legacy files and development notes; excluded from the
  published Jekyll site.
- `_site/`: local build output generated by Jekyll; not edited directly.

## Search Index Updates

The search pages do not query the CSV files directly in the browser. They use
prebuilt JSON files under `assets/`.

To update **CNKI Search** after editing `files/cnki_list.csv`:

```bash
python3 scripts/build_cnki_search_static_data.py
bundle exec jekyll build
```

Commit the updated files under `assets/cnki-search/`.

To update **Econ Search** after editing `files/RePEc_list.csv`:

```bash
python3 scripts/build_econ_search_static_data.py
bundle exec jekyll build
```

Commit the updated files under `assets/econ-search/`. The Econ Search frontend
defaults to rank 1, 2, and 3 journals; lower-priority ranks are available only
when selected on the page.

## Local Development

Install dependencies once:

```bash
bundle install
```

Build the site:

```bash
bundle exec jekyll build
```

Preview locally:

```bash
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000/`.
