# Miscellany

This folder keeps legacy, development, and archive files out of the main site root.

It is excluded from the Jekyll build in `_config.yml`, so files here are preserved in
the repository but are not published as part of the generated website.

Current groups:

- `docs/`: archived project notes and template documentation.
- `generators/`: older helper scripts and notebooks for generating content.
- `legacy-api/`: previous backend API experiments.
- `legacy-econ-search/`: previous Econ Search source project and historical data.
- `node/`: legacy Node deployment/config files.
- `talkmap/`: old talk map scripts and generated map assets.
